# Tech Stack

## Core Technologies

- **Language:** TypeScript/JavaScript
- **Framework:** React Native
- **Platforms:** iOS and Android

## State Management

- **Redux:** Utilizing Redux Toolkit, Saga, and Thunk for robust state handling.

## Testing & Quality Assurance

- **Unit Testing:** Jest
- **E2E Testing:** Detox
- **Integration Testing:** Appwright
- **Performance Testing:** Reassure

## Styling & UI

- **Styling:** Tailwind CSS via `twrnc`.
- **Design System:** MetaMask Design System (MMDS) using design tokens.

## Infrastructure & Tools

- **Package Manager:** Yarn (v4)
- **Bundler:** Metro
- **Error Tracking:** Sentry
- **Backend Services:** Firebase
