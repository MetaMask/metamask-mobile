# Initial Concept

MetaMask Mobile is a self-custodial multi-chain wallet and Web3 browser. It allows users to manage their digital assets, interact with decentralized applications (dapps), and securely store their private keys on their mobile devices.

# Product Definition

## Target Users

- **Crypto Enthusiasts:** Users who actively manage assets and interact with dapps across multiple chains.
- **Mainstream Users:** Individuals looking for a secure and accessible entry point into the decentralized ecosystem.

## Goals

- **Accessibility:** Provide a seamless mobile experience for interacting with the decentralized web.
- **Security:** Ensure the highest level of security for user keys and assets through self-custody.
- **Multi-Chain Interoperability:** Support a wide range of networks including Ethereum (EVMs), Bitcoin, Solana, and others.
- **Extensibility:** Enable third-party developers to extend wallet functionality securely via the Snaps framework.

## Core Features

- **Multi-Chain Wallet Management:** Manage accounts for Ethereum, Bitcoin, Solana, and other supported chains.
- **MetaMask Snaps:** A system that allows users to add new features and functionality (e.g., security insights, notification services, and support for additional protocols) directly to their wallet.
- **Dapp Browser:** A built-in browser for discovering and interacting with decentralized websites.
- **UI & Component Architecture:**
  - **MMDS (MetaMask Design System):** The primary and preferred design system for all new UI development, ensuring visual consistency and accessibility.
  - **Legacy Design System:** Supporting existing features while transitioning toward MMDS.
- **Network Management:** Built-in support for major chains and the ability to add custom RPC networks.
