# Security Policy

## Reporting a Vulnerability
If you discover a security vulnerability within MetaMask Mobile, please do not open a public issue. Instead, please report it through our official bug bounty program at [HackerOne](https://hackerone.com/metamask).

## Supported Versions
We only provide security updates for the latest stable version of the app available in the App Store and Google Play Store.

## Responsible Disclosure
We appreciate your help in keeping MetaMask safe. We will acknowledge your report and provide a timeline for resolution.
